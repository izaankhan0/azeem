My Services

Graphic Designing

Creating visual content using software like Adobe Photoshop and Illustrator. Involves design elements like typography, imagery, and layout techniques for branding and digital media.

Computer Aided Design (CAD)

CAD uses software to create accurate 2D and 3D models for engineering, architecture, and manufacturing. It streamlines design, aiding in precise technical drawings and reducing errors before production

Python

A versatile programming language used in web development, data analysis, and AI. Known for simplicity and vast libraries for scripting and complex applications.
